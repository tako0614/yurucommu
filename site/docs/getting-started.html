クイックスタート

5分でYurucommuをセットアップしてデプロイする手順です。

前提条件

  • Node.js 18以上
  • Cloudflareアカウント（無料プランで可）
  • Git

Step 1: リポジトリをクローン

git clone https://github.com/example/yurucommu.git
cd yurucommu
npm install

Step 2: Cloudflare CLIでログイン

npx wrangler login

ブラウザが開くので、Cloudflareアカウントで認証してください。

Step 3: D1データベースを作成

npx wrangler d1 create yurucommu-db

作成後、表示される database_id をメモしてください。

Step 4: R2バケットを作成

npx wrangler r2 bucket create yurucommu-media

Step 5: wrangler.tomlを設定

wrangler.tomlを開き、D1のIDを設定します：

[[d1_databases]]
binding = "DB"
database_name = "yurucommu-db"
database_id = "ここにStep 3でメモしたIDを入力"

Step 6: データベースマイグレーション

npx wrangler d1 migrations apply yurucommu-db --remote

Step 7: デプロイ

npm run build
npx wrangler deploy

デプロイが完了すると、URLが表示されます（例: yurucommu.xxx.workers.dev）

Step 8: 初期設定

デプロイしたURLにアクセスし、初回セットアップを行います：

  1. ユーザー名を設定
  2. 表示名を設定
  3. パスワードを設定（AUTH_MODE=passwordの場合）

次のステップ

  • カスタムドメインを設定する
  • 環境変数を詳しく設定する
  • 最初の投稿をする
