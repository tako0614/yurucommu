最初の投稿

Yurucommuで投稿を作成し、Fediverseに発信する方法を説明します。

投稿の作成

  1. タイムライン画面の上部にある投稿欄にテキストを入力
  2. 「投稿」ボタンをクリック

画像・動画の添付

  1. 投稿欄の下にある画像アイコンをクリック
  2. ファイルを選択（複数選択可、最大4ファイル）
  3. アップロードが完了したら「投稿」をクリック

対応フォーマット：

  • 画像: JPEG, PNG, GIF, WebP
  • 動画: MP4, WebM

ストーリー

24時間で消える短い投稿（ストーリー）を作成できます。

  1. タイムライン上部のストーリーバーで「+」ボタンをクリック
  2. 画像または動画をアップロード
  3. 表示時間を設定（3秒〜15秒）
  4. テキストオーバーレイを追加（任意）
  5. 「Share」をクリック

公開範囲

投稿の公開範囲は以下から選択できます：

範囲 説明
公開 誰でも閲覧可能、公開タイムラインに表示
未収載 URLを知っている人は閲覧可能、公開タイムラインには非表示
フォロワー限定 フォロワーのみ閲覧可能
ダイレクト メンションした相手のみ閲覧可能

メンション

他のユーザーに通知を送りたい場合は、メンションを使用します：

@username@example.com こんにちは！

ハッシュタグ

投稿にハッシュタグを付けると、同じハッシュタグの投稿を検索できます：

今日の #yurucommu の設定完了！ #activitypub

投稿の削除

  1. 削除したい投稿のメニュー（...）をクリック
  2. 「削除」を選択
  3. 確認ダイアログで「削除」をクリック
注意: 削除した投稿は、連合先のサーバーにも削除リクエストが送信されますが、キャッシュされている場合は完全に削除されない可能性があります。
