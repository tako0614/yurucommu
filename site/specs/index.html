ActivityPub拡張仕様

YurucommuはActivityPubプロトコルを拡張して、コミュニティ機能やその他の独自機能を実装しています。 これらの仕様は、他の実装との相互運用性を確保するために公開されています。

仕様のステータス

各仕様は以下のステータスを持ちます：

  • Draft - 策定中。大きな変更の可能性あり
  • Experimental - 実験的実装。フィードバック募集中
  • Stable - 安定版。後方互換性を維持

名前空間

Yurucommuの拡張プロパティは以下の名前空間を使用します： 

{
  "@context": [
    "https://www.w3.org/ns/activitystreams",
    "https://yurucommu.com/ns"
  ]
}

実装ガイドライン

Yurucommuの拡張仕様を実装する際は、以下のガイドラインに従ってください：

  1. 下位互換性 - 標準のActivityPub実装と互換性を維持する
  2. オプショナル - 拡張機能は必須ではなく、サポートしない実装でも基本機能は動作する
  3. 名前空間の使用 - 拡張プロパティには必ず名前空間を指定する
  4. ドキュメント - 実装の挙動を文書化し、テストケースを提供する
